## Purpose

These documents are preserved for:

1. **Historical reference** - Understanding why certain decisions were made
2. **Debugging** - If similar issues arise, these provide context
3. **Onboarding** - New contributors can learn from past implementations
4. **Documentation** - Some content may be incorporated into formal docs
