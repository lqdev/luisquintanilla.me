# Demo and Test Files Archive

This directory contains demo files, test artifacts, and prototype files that were used during development.

## Contents

| File | Description |
|------|-------------|
| blogroll-demo.html | Prototype HTML for blogroll feature |
| rome-favorites-demo.gpx | Sample GPX file for travel guide feature |

## Purpose

These files were created for:

1. **Feature prototyping** - Testing feature ideas before implementation
2. **Integration testing** - Validating external integrations
3. **Documentation examples** - Showing expected formats

## Why Archived

These files:
- Are not needed for production builds
- Cluttered the root directory
- May be useful for future reference
- Document expected data formats
