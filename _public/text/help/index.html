About This Site

This is a text-only, accessibility-optimized version of Luis Quintanilla's website. It's designed to work on any device and connection, including:

  • 2G mobile networks and slow connections
  • Flip phones and basic mobile browsers
  • Screen readers and assistive technologies
  • Devices with limited memory or processing power
  • Users who prefer minimal, distraction-free interfaces

Navigation Tips

  • Use Tab and Shift+Tab to navigate between links
  • Press Enter to follow a link
  • Use arrow keys to scroll on most devices
  • Skip links are available for screen reader users

Content Organization

Content is organized by type:

  • Posts: Blog articles and longer content
  • Notes: Short updates and microblog content
  • Responses: Replies and bookmarks to other content
  • Snippets: Code examples and technical notes
  • Wiki: Knowledge base and reference material
  • Presentations: Slides and presentation materials
  • Reviews: Book reviews and recommendations
  • Albums: Photo collections and visual content

RSS Feeds

RSS feeds are available for all content types at /text/feeds/. These feeds contain the same content as the full site feeds.

Full Website

To access the full website with graphics and interactive features, visit the main site.

Accessibility Features

  • Semantic HTML structure for screen readers
  • High contrast text and background colors
  • Keyboard navigation support
  • Skip links for efficient navigation
  • WCAG 2.1 AA compliance
  • Works without CSS or JavaScript

Performance

This site is optimized for minimal data usage. Most pages load in under 50KB, making them suitable for very slow connections and limited data plans.
