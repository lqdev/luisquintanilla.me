About This MVP

This project demonstrates how to rearchitect complex static site generators using the Unix philosophy. Instead of a monolithic application with heavy dependencies, we use focused tools that do one thing well:

  • pandoc - Markdown to HTML conversion
  • yq - YAML processing
  • envsubst - Template substitution
  • make - Build orchestration
  • bash - Glue logic and scripting
