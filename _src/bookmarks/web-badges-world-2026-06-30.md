> Web Badges World has been completely rewritten using a modern web framework and cloud infrastucture, providing drastically improved performance, while at the same time sporting a new retro aesthetic! It also includes new features like a [web badge generator](https://web.badges.world/#) so you can create and download your own web badges, and optionally submit them to be included in this archive. If you happen to have an existing web badge or a collection of web badges and would like to add them to the site, you can [upload them here](https://web.badges.world/#).  
> <br>
> It is also quite simple to make your own web badges using even a basic image editor. The font commonly used on web badges is called [Silkscreen](https://kottke.org/plus/type/silkscreen/), and can be downloaded and used for free under the [Open Font License](http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&id=OFL). The only real rule is that your file must be exactly 80x15 pixels, but you can follow the guidelines in the picture below for the standard web badge look.
